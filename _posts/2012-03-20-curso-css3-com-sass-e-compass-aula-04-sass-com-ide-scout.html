Vídeo/Screencast de hoje: aula 04 do curso gratuito de CSS3 com Sass e Compass!

Na aula de hoje vamos aprender como compilar o arquivos SASS em um arquivo CSS, e também vamos aprender a 'vigiar' um diretório para gerar os arquivos CSS automaticamente se alguma mudança acontecer em algum arquivo SASS. Só que ao contrário da aula passada, onde vimos isso sendo feito com linha de comando, hoje vamos aprender como usar a IDE Scout para fazer isso de forma mais automática! Também vamos aprender quais são as vantagens de usar a IDE Scout em vez de usar a linha de comando.

Quem quiser ver todas as aulas já publicadas, é só clicar aqui.

Espero que gostem dessa aula e fico no aguardo do feedback de vocês! :)

Requerimentos:

  • Ruby
  • Gem Sass
  • Editor de Texto. Nas aulas estou usando o Sublime Text 2.
  • Scout (Windows e Mac) - sorry, não tem pra Linux
