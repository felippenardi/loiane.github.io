Oi pessoal,

O post de hoje é uma dica rápida sobre como mudar a versão do XCode caso você tenha mais de uma versão instalada.

Com o lançamento beta do iOS 7 para desenvolvedores, baixei o XCode 5 (e devo dizer que estou gostando mais da versão 5 do que da versão 4 viu!).

E agora com o lançamento do PhoneGap 3 (em breve artigos aqui no blog sobre esse assunto!) e com esse lançamento uma nova interface via linha de comando. Neste caso o PhoneGap 3 só funciona por enquanto com o XCode 4, pois a versão 5 ainda é beta e só está disponível para os desenvolvedores certificados pela Apple.

Por esse motivo, preciso ficar mudando entre uma versão e outra. Uso o XCode 5 para brinca com iOS 7 e XCode 4 para brincar com Phonegap 3.

Para isso, podemos usar o comando xcode-select, que gerencia a localização do diretório do XCode para várias ferramentas de build.

Antes de instalar o XCode 5, se eu usar o comando abaixo

[code lang="bash" firstline="1" toolbar="true" collapse="false" wraplines="false"]$ xcode-select --print-path[/code]

Vou ter o seguinte output:

[code lang="bash" firstline="1" toolbar="true" collapse="false" wraplines="false"]/Applications/Xcode.app/Contents/Developer[/code]

Depois que instalei o XCode 5, se usar o mesmo comando o output será:

[code lang="bash" firstline="1" toolbar="true" collapse="false" wraplines="false"]$xcode-select --print-path
/Applications/Xcode5-DP.app/Contents/Developer[/code]

O parâmetro -switch fala para o xcode-select qual o caminho do XCode que você deseja setar como padrão. Se depois que você tiver o XCode 5 instalado quiser voltar pro XCode 4 ou vice versa, basta usar o comando abaixo:

[code lang="bash" firstline="1" toolbar="true" collapse="false" wraplines="false"]$ sudo xcode-select -switch /Applications/Xcode.app[/code]
