Ei pessoal,

Postei aqui a atualização de como criar um projeto PhoneGap para iOS e hoje vou postar a atualização para Android. Já postei aqui o tutorial do Phone Gap 1.x para Android. O tutorial continua valendo, mas na versão 2.x se tornou um pouco mais simples criar o projeto.

Requisitos:

  • Eclipse IDE (qualquer plataforma)
  • Android SDK
  • ADT Plugin para Eclipse
  • Phone Gap SDK
Para esse tutorial, também é requisito já ter o ambiente Android todo configurado, ou seja, ter o SDK instalado na sua máquina e o plugin para Eclipse instalado também. Para maiores informações veja os dois links a seguir:
  • http://developer.android.com/sdk/installing.html
  • http://developer.android.com/sdk/eclipse-adt.html#installing
Precisa também ter instalado Intel x86 Atom System Images e Intel Hardware Accelerated Execution Manager (debaixo de Extras no gerenciador do Android SDK).

Passo a passo:

  1. Criar uma aplicação Android usando linha de comando
  2. Abrir o projeto no Eclipse
  3. Executar no Emulador Android
  4. Próximos Passos

1 - Criando um novo Projeto Android usando Linha de Comando

Antes de começar, precisamos setar a variável de ambiente do Android no sistema, se já não foi feito. Se você usa Windows, é bem similar como setar o JAVA_HOME. Para maiores detalhes, olhe no link de referência no final do post.

Com o Phone Gap unzipado, abra o terminal e troque o diretório para suaInstalaçãoPhoneGap/lib/android/bin

Depois digite o comando para criar o projeto:

[code lang="bash" firstline="1" toolbar="true" collapse="false" wraplines="false"]./create /Users/loiane/development/phonegap-projects/HelloWorldAndroid com.loiane.helloworld HelloWorldAndroid[/code]

Onde

  1. ./create é o comando para criar o projeto
  2. /Users/loiane/development/phonegap-projects/HelloWorldAndroid é o caminho onde o projeto vai ser criado
  3. com.loiane.helloworld nome do pacote
  4. HelloWorldAndroid nome do projeto

E o projeto será criado:

2 - Abrindo o Projeto no Eclipse

Abra o Eclipse. Vá em New -> Other e escolha Android Project from Existing Code
