Olá pessoal,

Vídeo/Screencast de hoje: aula 07 do curso gratuito de Java!

Na aula de hoje vamos aprender que errar é humano! E também podemos cometer alguns erros enquanto programamos com Java, o que é normal. O ponto é: o que fazemos para corrigir nosso programa quando esses erros acontecem e que tipo de erros podem acontecer!

Quem quiser ver todas as aulas já publicadas, é só clicar aqui.

Espero que gostem dessa aula e fico no aguardo do feedback de vocês! 

Requerimentos:

  • Ter o JDK do Java Instalado (vc pode conferir a instalação aqui: Windows, Linux Ubuntu ou Mac OS)
  • Editor de Texto (use o seu favorito, uso o Sublime Text nas aulas – tem pra Windows, Linux e Mac OS e é grátis)
  • Linha de Comando (terminal/cmd)
