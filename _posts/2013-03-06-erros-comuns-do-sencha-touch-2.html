Ei pessoal,

Muita gente posta erros do Sencha Touch 2 aqui no blog e no fórum brasileiro também. Resolvi então escrever esse post com algumas dicas rápidas que respondem à maioria das perguntas que recebo.

Pré-requisitos para usar Sencha Touch 2

Precisa ter instalado na máquia:

  • Java JDK - e ter as variáveis de ambiente configuradas
  • Ruby
  • Sass e Compass

Se você não tem algum desses items acima e está tentando usar comandos do Sencha SDK Tools ou Sencha Command, instale e tente de novo.

Versão do Sencha Touch 2 x Sencha SDK Tools / Sencha Command

O pessoal faz uma confusão na hora de usar esses dois juntos. Vou colocar o mapeamento aqui. Tem que usar o mapeamento correto viu, senão dá erro:

  • Sencha Touch 2.0.x - Sencha SDK Tools (a aula 3 do curso de Sencha Touch 2 aqui do blog usa esse)
  • Sencha Touch 2.1.x - Sencha Command 3.0.x
  • Sencha Touch 2.2.x - Sencha Command 3.1.x

Sencha SDK Tools x Sencha Command: Comandos diferentes

Os comandos mudaram de uma versão para outra. Eis alguns comandos do Sencha Command

No Sencha Command, para gerar app usa: sencha generate app NomeApp DiretorioDaApp e não sencha app generate

Para fazer build usa: sencha app build - e gera o build de produção

Para empacotar usa: sencha package run packager.json
