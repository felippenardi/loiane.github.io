Oi Pessoal,

Volta e meia posto alguma coisa sobre desenvolvimento mobile aqui no blog, além do Curso Gratuito de Sencha Touch. E o Android hoje é a plataforma mais popular, devido a quase 70% de market share. Por esse motivo, tem muita gente com target para Android.

Como posto mais sobre desenvolvimento híbrido, e por mais que desenvolvimento híbrido use tecnologias web, saber o basicão de desenvolvimento nativo é necessário, pois ajuda muito na hora de debugar, fazer build para as app stores, etc. E é claro, ter o seu computador preparado para começar o desenvolvimento é um requisito.

Por esse motivo, resolvi escrever esse post, pois pode ser de muita ajuda para quem quer começar a desenvolver e não não sabe por onde começar.

Uma dica: esse post vai tratar apenas sobre a configuração do seu computador e um hello world pra testar pra ver se tudo está ok. Mas se você quiser mesmo investir e aprender a como desenvolver para Android, é necessário também saber pelo menos Java Básico! :)

Fiz um vídeo mostrando passo a passo. Além da instalação, também mostro como gerenciar e criar os emuladores, ou seja, todos os passos para você começar a mergulhar nesse mundo do Android.

No vídeo menciono 3 outros vídeos de como instalar e configurar o Java. Essa é a única coisa diferente entre Windows, Linux e Mac, por isso tenho esses vídeos separados, assim você só precisa assistir o vídeo para o seu sistema operacional. Como a instalação e configuração do Java é obrigatória e um requerimento para o ambiente Android, recomendo assistir e seguir os passos dos vídeos abaixo antes de seguir os passos do vídeo desse post.
