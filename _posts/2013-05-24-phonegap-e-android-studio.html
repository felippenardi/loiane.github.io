Ei pessoal,

No dia 15 de maio (semana passada) aconteceu o Google IO, um dos eventos mais importantes da Google, onde eles fazem anúncios e mostram as novidades, e uma das novidades foi o anúncio de uma nova IDE oficial para o desenvolvimento de aplicações Android que é o Android Studio. Hoje, o ambiente "oficial" é o Eclipse com o plugin do Android.

Bem, conforme já publiquei alguns tutoriais aqui no blog, a partir a versão 2.x do phonegap, o framework oferece um recurso que já cria um projeto inicial Android/PhoneGap para ser executado no eclipse. Mas como agora a IDE será o Android Studio, as coisas serão um pouquinho diferentes, mas nada que a gente precise se preocupar, apenas a IDE é diferente.

Mas como fica a questão dos projetos gerados pelo phonegap? A versão atual do phonegap é a 2.7, mas logo logo eles devem lançar uma nova versão que deve dar 100% de suporte ao Android Studio. Enquanto isso, se quiserem testar, podem seguir esse tutorial.

Lembrando que o tutorial de hello world oficial continua valendo também e você pode ler aqui.

1 - Primeiro a gente cria um projeto com o PhoneGap. Vou criar um projeto chamado AndroidStudioPhoneGap. Esse passo continua o mesmo.

2 - Depois a gente precisa abrir o Android Studio. Na tela inicial vamos escolher Import Project:

3 - O próximo passo é escolher o projeto que foi gerado pelo PhoneGap e clicar em OK:

4 - Depois vamos começar o processo de import e precisamos fazer algumas configurações. A primeira é escolher a opção "Create project from existing sources" e clicar em Next:

5 - E o próximo passo será especificar o nome do projeto e o diretório do projeto. Uma observação muito importante é que o diretório deve ser o mesmo que escolhemos na hora de usar o comando create do phonegap. Não queremos mudar o diretório.
