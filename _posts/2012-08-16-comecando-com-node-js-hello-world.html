Ei pessoal,

De vez em quando sinto a necessidade de estudar assuntos diferentes para aumentar meu conhecimento sobre assuntos diferentes e também não ficar "boiando" em eventos de TI. O assunto que escolhi é o Node.JS, o javascript no lado servidor!

A gente sempre ouve que JavaScript é a linguagem do browser, linguagem cliente, front-end, mas qual é o problema em usar JavaScript no backend também? Bem, como vamos executar o JavaScript no servidor, precisamos primeiro interpretá-lo e depois executá-lo. É isso que o Node.JS faz, através do uso da VM (Virtual Machine) V8 do Google, o mesmo ambiente que o Google Chrome usa.

Instalação

Para instalar o Node.js é muito simples. Basta fazer o download a partir desse link: http://nodejs.org/#download

Quem usa Windows ou Mac OS, é só baixar o instalador, dar next, next, finish e tá pronto! Quem usa Linux pode usar um gerenciador de pacotes.

Hello World

Para fazer o Hello, World é bem simples: abra o seu editor de texto favorito, digite:

[code lang="js" firstline="1" toolbar="true" collapse="false" wraplines="false"]
console.log("Hello World");
[/code]

Salve como helloworld.js.

Agora abra o console/terminal, mude para o diretório onde está o arquivo que acabamos de criar e digite:

[code lang="bash" firstline="1" toolbar="true" collapse="false" wraplines="false"]
node helloworld.js
[/code]

E o resultado será:

Pronto! Fizemos a nossa primeira aplicação com Node.js!
