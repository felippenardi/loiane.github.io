Quer participar do processo de certificação do JDK7?

A SUN tem feito o job task analysis (JTA) dos papéis de um programador e desenvolvedor do JDK 7. O JTA irá ajudar na criação definição dos objetivos dos exames:
Sun Certified Java Programmer (SCJP) e
Sun Certified Java Developer (SCJD)

Um programador está em um nível de iniciante e um desenvolvedor está em um nível avançado.

A SUN precisa de alguns minutos do seu tempo para validar os JTAs para ambos os papéis

O que você precisa fazer:

  1. Faça o login em Http://wikis.sun.com/display/JTASurvey utilizando seu Sun Online account ID e senha
  2. Leia as orientações na home page
  3. Revise o Java SE programmer JTA e o Java SE developer JTA
  4. Mapeie a lista de funções/tarefas e atividades para competências das tarefas do dia-a-dia de um programador/desenvolvedor Java SE
  5. Forneça comentários específicos. Não se mencione se certos temas aparecerem obsoletos, ou se determinadas novas tarefas devem ser acrescentadas
  6. Por favor, vá em frente e edite o documento com as suas adições. Garanta que sua adição/comentário esteje formatada em uma cor diferente
  7. Concluir o formulário

Obs: Se você não tiver uma conta, você pode criar uma no site.

Você pode influenciar as certificações JDK 7 fazendo uma revisão das JTAs.
O site permanecerá online por uma semana ou duas, então, melhor participar agora antes que seja tarde.
