Olá Pessoal,

Hoje decidi responder uma pergunta que muita gente me faz: o que eu uso para gravar as aulas e screencasts aqui do blog?

São basicamente 3 coisas:

  • Meu MacBook Pro de guerra
  • Phone de ouvido do iPhone
  • Screenflow

Hardware

Bem, para começar, você precisa de um computador! Eu uso meu MacBook Pro. Às vezes uso o iMac, mas ainda prefiro gravar no MacBook.

A questão do phone de ouvido é simples: é o único que eu tenho! rs Quando comprei meu iPhone ele veio junto e me apaixonei por esse phone (sei que muita gente não gosta dele, mas eu adoro!). E ele funciona super bem nos computadores da Apple (pq será?). Sei que a qualidade dele não é das melhores, mas por enquanto tem me atendido. Eu queria muito um microfone usb, mas até hoje não achei um bom que funcione no Mac OS. O ruim dos computadores Apple é isso, eles vem só com um furinho de entrada para phone de ouvido/microfone; não é como nos PCs que vem com um furinho para phone e outro para microfone. Quem tiver alguma dica, aceito, viu!

Screenflow

E por último, quem faz toda a mágica: Screenflow! Esse software para gravação de screencasts é maravilhoso! E é o mais usado no mundo Mac OS também. O lado ruim é que é pago - custa US$ 99 Obamas (cerca de 200 Dilmas), mas vale cada centavo viu! Logo que comprei meu Mac e queria gravar uns vídeos para o blog, procurei uma ferramenta, e todos os meu amigos que tem Mac me indicaram essa. Só que eu lutei muito para tirar 99 dólares do meu bolso e comprar. A gente fica tão costumado a piratear software quando usa Window que qualquer coisa fica caro depois. Mas depois que comecei a usar Mac aprendi a comprar software.

Uma alternativa é o Camtasia Studio. Quando eu usava Windows usava ele. É uma ótima ferramenta para gravar screencasts para quem usa Windows, mas no mundo dos Macs ainda é bem novo, por isso decidi comprar o Screenflow também.

Screenflow é mais mais estável e tem um mercado de usuários maior também. E gente, como facilita a vida! É a coisa mais fácil do mundo editar as aulas! Eu não faço nada de especial, apenas corto algumas partes que eu fico pensando muito, ou fico tossindo (minha garganta não é das melhores), ou faço uma parada para tomar água.

Depois de editar, é só exportar para .mov e fazer upload para o Vimeo e YouTube. Dá para fazer o upload diretamente da ferramenta também, mas prefiro exportar para depois upar.

Omnigraffle

Bom, essa é uma ferramenta extra que uso para fazer os diagramas e desenhos tanto para as aulas quanto aqui para o blog. Esse Omnigraffle é uma das melhores ferramentas que já usei na vida. É uma alternativa ao Microsoft Visio para Windows. A Microsoft só fez o Word, Excel, Power Point e Outlook para Mac.

Só que na minha opinião, o Omnigraffle faz muito mais coisas que o Visio. E ficam bem mais bonitinhas também! :)

Infelizmente, também é pago - lá se vão mais US$ 99 Obamas (200 Dilmas), mas também vale a pena. Tem a versão Pro também, que custa US$ 199 Obamas, mas a versão padrão já faz tudo o que eu preciso.

O mapa mental do ExtJS 4 que fiz aqui para o blog fiz nessa ferramenta. Dá para fazer download de stencils de graça nesse site. A ferramenta também é integrada a esse site, então dá pra procurar e fazer download diretamente do Omnigraffle.

Todos os diagramas que fiz para o meu livro também usei essa ferramenta. Só que a editora tem sua própria ferramenta e re-fez tudo de novo! :/

Bem pessoal, é isso! Até o próximo post! :)
