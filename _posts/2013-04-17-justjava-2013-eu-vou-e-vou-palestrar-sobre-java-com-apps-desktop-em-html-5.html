Ei pessoal,

Passando para divulgar mais um evento onde vou palestrar. Dessa vez será o JustJava que vai acontecer nos dia 7 e 8 de Junho de 2013 no SENAC - Campus Santo Amaro (mesmo lugar de 2012), em São Paulo.

Esse ano vou falar sobre aplicações desktop (nativa) feitas com HTML 5 (ou você também pode usar o seu framework javascript preferido como JQuery, Ext JS e outros) tendo o Java no backend. Essa é uma vertente nova que está surgindo no mercado e é sempre bom ficar ligado nas novidades né? O foco da palestra será no Java, pois precisamos fazer algumas mudanças no código para que funcione com aplicações assim.

Segue título e descrição da palestra:

Aplicações desktop nativas com HTML5, CSS3 e JS com backend usando Servlets 3

HTML 5 é a tecnologia do momento. Tem sido usada tanto em sistemas web quanto sistemas mobile. Uma novidade agora chega ao mercado: aplicações desktop nativas com front-end em HTML 5, CSS 3 e JS, mas no backend, você pode continuar a usar a sua linguagem favorita, como Java por exemplo. Nessa palestra vamos conhecer um pouco mais sobre essa nova vertente e como podemos utilizar o Java (e o que temos que mudar em aplicações atuais) para que funcione com aplicações desktop. Também vamos ver uma demo de uma aplicação com Java no servidor, e a mesma aplicação nativa rodando no Mac OS e no Windows (e apenas 1 código).
