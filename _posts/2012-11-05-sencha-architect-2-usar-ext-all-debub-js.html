Mais uma dica rápida sobre Sencha Architect 2.

Hoje vamos ver como faz para usar a versão debug do SDK do ExtJS 4 no lugar do ext-all.js (que é o arquivo padrão usando pelo Sencha Architect 2).

1 - Na estrutura do projeto, ache a parte de Resources -> Libray. Marque Debug e pronto!

2 - Agora vai carregar a versão debug do ExtJS!
