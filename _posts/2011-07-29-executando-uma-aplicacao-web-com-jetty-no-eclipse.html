Este tutorial tem como objetivo mostrar como executar aplicações web no Jetty webserver diretamente do Eclipse.

Requerimentos:

  1. Como instalar e executar o Jetty
  2. Instalar plugin Run-Jetty-Run no Eclipse

Como mencionei nos tutoriais listados acima, estou tentando usar Jetty no lugar do Tomcat. Neste post, vou mostrar como é simples e fácil fazer deploy de uma aplicação web no Jetty diretamente no Eclipse (com uso do plugin Run-Jetty-Run).

Steps:

1 – Crie um Projeto do tipo Dynamic Web Project (você pode clicar com o botão direito na view de Project Explorer ou pelo menu File -> New…):

2 – Dê um nome para o projeto web e clique em Next:

3 – Clique em Next:

4 – Marque a opção Generate web.xml… e clique em Finish:

5 – O projeto deve ficar assim - projeto bem simples - é preciso criar uma página index.html também:

6 – Selecione o projeto, clique no botão Run  -> Run Configurations…

7 – Dê um duplo clique na opção Jetty Webapp. Será criada uma configuração com o nome do projeto criado. Depois é só clicar em Run:

8 – Espere pelo servidor iniciar:

9 – Agora é só testar a aplicação!
