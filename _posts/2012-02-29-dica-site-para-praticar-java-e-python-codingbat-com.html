Recentemente descobri um novo site para praticar conceitos básicos de Java e Python para iniciantes. Apresento-lhes o condingbat.com!

O CodingBat é um site voltado para iniciantes que queiram praticar Java e/ou Python. O site é mantido por um professor da universidade de Stanford, uma das mais conceituadas mundialmente.

O site conta com problemas de Warmups (aquecimento), manipulação de Strings, lógica booleana, manipulação de arrays, recursão, loops, etc.

Os problemas possuem um pequeno enunciado, onde o problema é descrito com exemplo de entrada e saída esperada. Eles já fornecem um método e basta você implementar a lógica do método. E a resposta se a sua implementação está correta é dada na hora.

Os problemas são bem básicos, mas são ótimos para quem está iniciando na linguagem, ou quer passar o tempo ou ainda quer treinar TDD.
