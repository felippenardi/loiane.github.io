Fiz esse tutorial quando estava no último ano da faculdade para ajudar alguns colegas que nunca tinham trabalhado com a ferramenta.

A pedidos, resolvi fazer algumas mudanças nele e acrescentar algumas coisas e postar aqui no blog. Talvez também possa ajudar outras pessoas que estão iniciando com o My SQL.

O trabalho de faculdade que eu tinha que fazer era criar um modelo ER, depois criar as tabelas no banco My SQL e desenvolver uma pequena aplicação em java para manipular os dados destas tabelas.

Resolvi então, fazer toda a parte de banco de dados através das ferramentas gratuitas que a própria My SQL fornece. E este tutorial aborda como eu fiz, desde a instalação das ferramentas e do bando de dados até um exemplo básico de como usar as ferramentas My SQL.

Dividi o tutorial em quatro partes:
1. Como instalar o My SQL (Windows XP)
2. Como instalar o My SQL Tools (Administrator, Query Browser, Migration Toolkit, e System Tray Monitor)
3. Instalando o My SQL Workbench (utilizada para criar o ER e gerar o script)
4. Exemplo: como criar um novo schema utilizando My SQL Workbench e My SQL Query Browser
