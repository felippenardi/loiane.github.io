Vídeo/Screencast de hoje: aula 03 do curso online gratuito aqui do blog: Sencha Touch 2!

Na aula de hoje vamos aprender como gerar uma aplicação automaticamente com Sencha SDK tools, e também vamos aprender como executar a app para tests no browser, fazer um build otimizado para produção (para diminuir o número de requests, etc), e por último, empacotar a app para iOS e Android e emular para as respectivas plataformas.

A aula de hoje dá para a gente ter uma boa noção do que o Sencha Touch 2 é capaz. Nas próximas aulas vamos aprender mais sobre os componentes e sobre o sistema de classes do Sencha Touch 2. Falei isso na aula, mas vou deixar isso documentado aqui no post também. Teremos mais 1 ou 2 aulas antes de entrar nos componentes do ST2, por isso, é indispensável assistir as aulas dos módulos de Sistema de Classes, Loader e Data Package do Curso Gratuito de ExtJS 4, já que essa parte da API é compartilhada entre o ExtJS 4 e Sencha Touch 2, e não vou repetir as mesmas aulas aqui. Por isso, se você ainda não assistiu a essas aulas, pode assistir: são aproximadamente 10 horas de aulas e conceitos que vamos precisar aqui no curso de Sencha Touch 2.

Quem quiser ver todas as aulas já publicadas, é só clicar aqui.

Recomendo altamente aprender ExtJS 4 também, afinal parte da API do Sencha Touch 2 é compartilhada com ExtJS 4. Você pode acompanhar o curso gratuito de ExtJS 4 aqui.

Espero que gostem dessa aula e fico no aguardo do feedback de vocês! :)

Obs: quem tiver com problemas para rodar o exemplo no emulador, troque o logger do arquivo json para false ao invés de no. Ou copie o sdk/src para build/package/skd.

Requerimentos:

  • SDK Sencha Touch 2
  • Sencha SDK Tools
  • Editor de Texto
  • WebServer (Apache)
  • iOS Emulator e/ou Android Emulator
  • Smartphone ou tablet (opcional)
